# Command install 

    pip install checkov

## Instalacion de checkov en centos 
    yum install python
    yum install pip
    sudo yum remove python3-requests
    pip install checkov

## Instalacion de checkov en linux ubuntu
    sudo apt update 
    sudo apt install pip3
    sudo pip3 install checkov
    export PATH=$PATH:/home/ubuntu/.local/bin/
